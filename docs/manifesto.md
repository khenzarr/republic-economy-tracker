# Republic Economy Tracker - Manifesto

## Our Vision

**Building the most transparent, community-driven ecosystem explorer for the Republic blockchain.**

We believe that blockchain ecosystems thrive on transparency, accessibility, and community participation. Republic Economy Tracker exists to provide the infrastructure that makes the Republic network more discoverable, measurable, and accessible to everyone—from builders to validators to investors.

---

## Core Principles

### 🔓 Transparency First

**Open Source, Open Data**

Every line of code is public. Every metric is verifiable. Every algorithm is auditable. We believe that trust in Web3 comes from transparency, not promises.

- All code is MIT licensed and available on GitHub
- Metrics are calculated directly from on-chain data
- Scoring algorithms are open and documented
- No hidden data sources or proprietary calculations

### 🤝 Community-Driven

**Built by the Community, for the Community**

This platform exists to serve the Republic ecosystem. We welcome contributions, feedback, and governance from all stakeholders.

- Open contribution model
- Community project submissions
- Transparent approval process
- Governance-ready architecture

### ⚡ Infrastructure Contribution

**Supporting the Republic Network**

As active participants in the Republic ecosystem, we're committed to providing reliable infrastructure and tools that benefit the entire network.

- Direct RPC integration
- Public API access
- Open-source reference implementations
- Validator-friendly architecture

### 📊 Data Integrity

**Accurate, Real-Time, Verifiable**

We prioritize data accuracy and real-time updates. All metrics come directly from the blockchain—no intermediaries, no delays, no manipulation.

- Direct blockchain queries via CosmJS
- Automated hourly updates
- Verifiable on-chain data
- Transparent calculation methods
